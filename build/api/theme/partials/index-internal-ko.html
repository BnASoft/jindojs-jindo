JindoJS는 NAVER에서 제작한 JavaScript library입니다.

Jindo는 Javascript Application 제작시에 반복적으로 발생하는 로직을 컴포넌트화하여 결과물의 품질을 향상시키기 위한 목적으로 작성되었습니다. NAVER에서 서비스하고 있는 수많은 페이지안의 UI들이 Jindo를 사용하여 개발되었습니다.그런만큼 Jindo는 크로스 브라우징에 있어서 거의 완벽한 호환성을 입증하고 있습니다.

사용방법

JS 파일을 다운로드 받아 HTML 파일에 아래와 같이 삽입하여 사용합니다. 
<script type="text/javascript" src="jindo.js"></script>
